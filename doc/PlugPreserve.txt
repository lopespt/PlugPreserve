------------------------------------------------------------------------------
INTRODUCTION

    This plugin uses git stash before PlugUpdate starts update. After all
    updates, all changes are restored with "git stash pop".

------------------------------------------------------------------------------
COMMANDS

							 *PlugUpdatePreserve*
:PlugUpdatePreserve	Execute *PlugUpdate* preserving own changes.
